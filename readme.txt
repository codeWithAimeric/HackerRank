Pour lancer un seul fichier de test : 
-"scripts": {
    "test": "jest --testPathPattern='../absolutePermutation.spec.js'"
  },
-remplacer ../absolutePermutation.spec.js par le nom du fichier de test à lancer 
-lancer la commande : "npm test"

Pour lancer tout les fichier de test en même temps : 
-"scripts": {
    "test": "jest"
  },
-lancer la commande : "npm test"
